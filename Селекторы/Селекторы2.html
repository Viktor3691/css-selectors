Классы элементов в CSS

В предыдущих уроках мы выбирали элементы страницы по имени тега, задавая стили, к примеру, одновременно всем абзацам. На
        странице, однако, могут быть абзацы разных типов, к которым нужно применять разные стили. Для решения такой проблемы
        можно разные абзацы отнести к разным CSS классам.

        Чтобы отнести тег какому-нибудь классу, нужно этому тегу написать атрибут class, а в нем - придуманное вами название
        класса, состоящее из букв, цифр, символов подчеркивания и дефиса.

        Давайте посмотрим на примере. Сделаем абзацы с двумя типами классов - eee и zzz:

paragraph with class eee

paragraph with class eee

paragraph with class zzz

paragraph with class zzz 

Давайте теперь обратимся в CSS к абзацам с различными классами и зададим им какие-нибудь стили. К примеру, покрасим
        абзацы с классом zzz в красный цвет, а абзацы с классом eee - в зеленый.

        Для этого в CSS файле нужно обратиться к нашим классам. Обращение имеет следующий синтаксис: сначала идет символ точка,
        а затем придуманное нами имя класса. То есть для обращения к классу eee нужно написать .eee, а для класса zzz - написать
        .zzz.

        Итак, сделаем описанное. Добавим к нашему HTML еще и CSS стили для введенных нами классов:

Тег span для кусочков текста в HTML

Вы уже знаете, что для того, чтобы сделать некоторый кусочек текста, к примеру, жирным - следует использовать тег b.
        Однако, что делать, если я хочу покрасить кусочек текста в красный цвет? Для такого случая тега, подобного тегу b, не
        существует.

        Зато существует специальный тег span - сам по себе он абсолютно ничего не делает, но для него можно применить CSS стили,
        которые позволят нам добавить нужный эффект к кусочку текста.

        Давайте посмотрим на примере. Пусть у нас есть какой-то текст:

This is a paragraph with text. 

Пусть мы хотим слово "текст" сделать красным. Для этого обернем этот кусочек текста в тег span и этому тегу дадим
        какой-нибудь класс:

This is a paragraph with text.

Ну а теперь покрасим элемент с таким классом в красный цвет:

This is a paragraph with text.

Lorem ipsum dolor sit amet.

Lorem ipsum dolor sit amet.
